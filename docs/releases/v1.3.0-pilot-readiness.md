# v1.3.0 — Pilot Readiness

**Release date:** May 6, 2026
**Cycle:** Pilot-readiness follow-up to v1.2 (`docs/remediation/agentic-plan-v1.2.md`)
**Plan:** `docs/remediation/agentic-plan-v1.3.md`
**Verdict at the time:** PILOT-READY pending follow-ups (which became v1.4)

This release closed five follow-up findings from a post-merge review of
the v1.2 squashed merge (PR #14). Three were partial v1.2 closures and
two were missed gaps. The cycle's process discipline change — one PR
per gap, no squash-merge — is itself one of the deliverables.

## Process discipline change

This cycle landed every gap as a separate PR with a distinct merge commit on `main` (PRs #15, #16, #17, #18, #19, #20 — six merge commits, six gaps). The contrast to v1.2's single squashed merge (PR #14, all 22 gaps in one commit) was deliberate and is now codified in `CONTRIBUTING.md` as the rule for all future remediation cycles.
